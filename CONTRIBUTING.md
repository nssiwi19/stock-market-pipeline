# Contributing Guide

## Branching Rule

- Khong push truc tiep vao `main`.
- Tao nhanh feature/fix branch theo mau: `feat/...`, `fix/...`, `chore/...`.
- Merge vao `main` qua Pull Request.

## Review Rule

- Moi PR can it nhat 1 reviewer.
- PR chi duoc merge khi workflow `Quality Gate` pass.

## Commit Hygiene

- Khong commit script debug tam thoi vao root repo.
- Khong commit secrets, log, output artifacts.
